本项目实现了一个主打校园社交的程序，满足了当前很多同学乃至老师的需求（同学比如竞赛组队，吃喝拼团，老师则有一起乘坐公交车等需求）。
本项目的特点是因为WebSocket,只要从电脑上下载了该程序，和服务器处于同一局域网并且服务器启动，就能实现实时的交互。

经调试，能无差错地顺利执行

实现环境：Android Studio,IntellijJ IDEA,MySQL80,Navicat
编程语言：Java、xml
采用框架：WebSocket、Fragment、Navigation、Menu、RecycleView、Adapter 

更多细节见本目录下的详细说明.docx以及演示视频.mp4
